============== Агрегатные функции

Вычислить среднее значение столбца col-name в таблице table-name.
SELECT AVG(col-name) FROM table-name;

Вычислить среднее значение столбца col-name и перевести в доллары х65 в таблице table-name.
SELECT AVG(col-name*65) FROM table-name;

Вычислить сумму всех значений столбца col-name в таблице table-name.
SELECT SUM(col-name) FROM table-name;

Найти максимальное и минимальное значение столбца col-name в таблице table-name.
SELECT MAX(col-name), MIN(col-name) FROM table-name;

Посчитать количество строк в таблице table-name.
SELECT COUNT(*) FROM table-name;

Вывести в одну строку через запятую значения всех ячеек столбца col-name в таблице table-name.
SELECT LIST(col-name) FROM table-name;
